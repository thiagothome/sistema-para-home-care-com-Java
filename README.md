# Sistema para Home Care

Este projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) e visa criar um sistema para Home Care utilizando tecnologias Java, JSF PrimeFaces, MySQL e ferramentas de modelagem de dados.

## Tecnologias Utilizadas

- **Java**: A linguagem de programação Java foi escolhida devido à sua robustez, portabilidade e ampla adoção na indústria de desenvolvimento de software.
- **JSF PrimeFaces**: O framework JavaServer Faces (JSF) em conjunto com o PrimeFaces foi utilizado para desenvolver a interface do usuário (UI) do sistema, proporcionando uma experiência de usuário rica e interativa.
- **MySQL**: O sistema de gerenciamento de banco de dados relacional MySQL foi escolhido para armazenar e gerenciar os dados do sistema, garantindo eficiência e confiabilidade no armazenamento e recuperação de informações.
- **Astah e BRModelo**: Foram utilizadas ferramentas de modelagem UML e de banco de dados para auxiliar na visualização e modelagem do sistema.

## Desenvolvimento

O projeto foi desenvolvido utilizando a IDE Eclipse e seguiu as práticas recomendadas de desenvolvimento de software, incluindo modularização do código e controle de versão com Git.

## Resultados

O sistema desenvolvido demonstrou ser uma solução eficaz e escalável para o gerenciamento de Home Care, oferecendo uma interface amigável e funcionalidades abrangentes para atender às necessidades dos profissionais de saúde e dos pacientes.
